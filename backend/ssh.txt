## Step 9: Create System Service

```bash
nano /etc/systemd/system/msae-backend.service
```

Paste this exact content:
```ini
[Unit]
Description=MS Ayub Enterprise Backend
After=network.target

[Service]
Type=simple
User=root
WorkingDirectory=/root/ms_ayub_enterpirse/backend
ExecStart=/root/ms_ayub_enterpirse/backend/msae-backend
Restart=always
RestartSec=10
StandardOutput=journal
StandardError=journal

[Install]
WantedBy=multi-user.target
```

Save with: `Ctrl+X`, then `Y`, then `Enter`

## Step 10: Start the Service

```bash
systemctl daemon-reload
systemctl start msae-backend
systemctl enable msae-backend
systemctl status msae-backend
```

## Step 11: Open Firewall

```bash
# Check if firewalld is running
systemctl status firewalld

# If running, open port 8080
firewall-cmd --permanent --add-port=8080/tcp
firewall-cmd --reload

# OR if you want to just disable firewall (easier for testing)
systemctl stop firewalld
systemctl disable firewalld
```
